# 自然语义地图选点定位功能实施计划

## 需求分析

在故障信息的编辑页面（`otnfault_edit.html`）中，为经纬度字段提供地图选点功能。
具体要求：
1. **入口**：在经纬度输入框旁增加一个图标按钮。
2. **交互**：点击按钮，弹出一个地图选择窗口（使用系统中已有的 `/plugins/otnfaults/map/location/?q=` 类似视图）。
3. **初始化中心点**：
   * 如果故障选择了 A 和 Z 站点，地图初始化定位到连接两站点的区域/路径上。
   * 如果仅选择了 A 站点，定位到 A 站点。
4. **选点操作**：用户可以在弹窗地图中拖动准星（或者可拖动的 marker）进行定位，同时界面下方显示当前指向的经纬度。
5. **回填数据**：点击“确定”按钮后，弹窗关闭，选中的经纬度自动填入父窗口（故障编辑页）的 `interruption_longitude` 和 `interruption_latitude` 字段中。

## Implementation Plan

### 1. 前端模板修改 (`otnfault_edit.html` & `forms.py`)
- 在 `otnfault_edit.html` 中通过 JavaScript 找到这两个字段，并在它们旁边加上一个地图选择按钮。
- 点击按钮打开弹出窗口：
  ```javascript
  const aSiteId = document.getElementById('id_interruption_location_a').value;
  // 获取选择的 Z 站点列表
  const zSiteOpts = document.getElementById('id_interruption_location').options;
  let zSites = [];
  for(let i=0; i<zSiteOpts.length; i++) {
        if(zSiteOpts[i].selected) zSites.push(zSiteOpts[i].value);
  }
  let url = '/plugins/otnfaults/map/location/?picker=true';
  if (aSiteId) url += '&a_site=' + aSiteId;
  if (zSites.length > 0) url += '&z_sites=' + zSites.join(',');
  let popup = window.open(url, 'MapPicker', 'width=800,height=600,left=200,top=200');
  ```
- 接收地图页面的回调：
  ```javascript
  window.addEventListener('message', function(event) {
      if (event.data.type === 'OTN_LOCATION_PICKED') {
          document.getElementById('id_interruption_longitude').value = event.data.lng.toFixed(6);
          document.getElementById('id_interruption_latitude').value = event.data.lat.toFixed(6);
      }
  });
  ```

### 2. 地图视图传参 (`views.py` - `LocationMapView` / 其他地图视图)
- 修改 `LocationMapView`，当发现 `picker=true` 或类似的参数时：
  - 读取 `a_site` 和 `z_sites` 的坐标，将计算出的 bounding box 或者默认中心点传入模板 context。
  - 在 `unified_map.html` 渲染时能获得针对当前故障选择的这几个站点的坐标数据。

### 3. 地图选点组件 (`unified_map_core.js` 或新建 `mode`)
- 因为 `LocationMapView` 目前复用了 `unified_map.html`，可以新建一个选点模式，或者是如果 `picker=true` 时展示选点界面：
  - 地图中心固定一个“+”号准星，或者加一个能在地图上拖动的 marker。由于需求提到了拖动地图调整定位，固定准星，拖回时候更新下方经纬度是最好的。
  - 绑定 `map.on('move', function() { let center = map.getCenter(); })` 事件，实时更新下方的显示。
  - 提供底部/顶部的一个悬浮面板，包含经纬度以及“确认选定”和“取消”按钮。
  - 点击“确认选定”执行 `window.opener.postMessage({"type": "OTN_LOCATION_PICKED", lng: center.lng, lat: center.lat}, "*")` 并 `window.close()`。
